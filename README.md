## 🚀 Featured Projects

### 🪙 [Ammazza WebAR](https://tryon.ammazza.me/) — Browser-Native AR Jewelry Try-On
Real-time AR jewelry try-on built from scratch with **JavaScript → React.js**, **MediaPipe** face/hand landmark detection, and **Three.js** for precise 3D mesh placement on face, neck, and hands. Used by jewellers to deliver immersive in-browser try-on experiences — zero app download required.

---

### 🏗️ Hedral — 3D AEC Platform
Led end-to-end frontend development of a complex architectural design platform for warehouses and building complexes. Built custom **Three.js loaders**, **GLSL WebGL shaders** for physically accurate materials, and integrated **Python GraphQL APIs** + **Mapbox** for real-time geolocated 3D building rendering. Architected state management with **Zustand + Redux + TanStack Query** across a team of 5 developers.

---

### 🛋️ Furniture AI — AR Interior Design Tool
Built an interactive interior design app overlaying **3D furniture models** onto room images using **Three.js**. Unified **Konva.js** (2D rendering), **OpenCV.js** (wall repainting via flood fill), and deep learning-based object removal — all on a single canvas.

---

### ⚙️ PCS Viewer — Enterprise CAD Configurator
Production 3D configurator library for complex CAD models with **explode view, transform handles, and keyframe animation controls**. Devised a custom lazy-loading algorithm achieving **75% reduction in heavy model load times**. Migrated infrastructure to **Azure Cloud** with .NET Core microservices.

---

### 🏠 ARHome — Architectural 3D Renderer
High-performance renderer for 3D building and house models, capable of handling complex objects **exceeding 100MB** while maintaining smooth interactivity.

---

### 🌐 Causeverse — Web AR/VR Collaborative Meeting Platform
Full-stack ownership of an immersive Web AR/VR meeting platform: **React + Three.js + Croquet.io** for real-time multi-user 3D rooms, **C# .NET** backend, admin panel for media asset management, and seamless API integration.

---

### 🔩 AI-APS Viewer — GPT-Powered CAD Query Interface
Node.js + Express.js API layer enabling natural language querying of **Autodesk APS (formerly Forge) buckets** via a GPT-based model — bridging conversational AI with 3D asset management.

---

### ☁️ SyncroDB — Cloud File Management System
Designed and built a cloud application for managing large file uploads via **signed URLs**, using **ASP.NET Core**, **Azure Storage + Compute**, **PostgreSQL**, and **Azure AD** for authentication.
